# 国土交通省 入札案件抽出 Chrome拡張機能

入札情報サービスの工事案件概要ページから必要な情報を自動的に抽出し、スプレッドシートに貼り付け可能な形式でクリップボードにコピーするChrome拡張機能です。

## 機能

- 入札案件の詳細情報を自動抽出
- スプレッドシート用にデータを整形
- ワンクリックでクリップボードにコピー
